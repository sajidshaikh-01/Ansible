| Flag           | Meaning                  |
| -------------- | ------------------------ |
| `-i inventory` | Inventory file           |
| `-m`           | Module name              |
| `-a`           | Module arguments         |
| `-b`           | Run as sudo (become)     |
| `-u ubuntu`    | SSH user                 |
| `-k`           | Prompt for SSH password  |
| `-K`           | Prompt for sudo password |
