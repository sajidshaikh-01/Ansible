⚠️ Important Points for Interview

| Topic                                    | Short Answer                                  |
| ---------------------------------------- | --------------------------------------------- |
| What keyword is used for conditions?     | `when`                                        |
| Can we use multiple conditions?          | Yes (list format preferred)                   |
| Can conditions be applied to handlers?   | Yes                                           |
| Order of task execution?                 | Conditions only determine skip/run, not order |
| Can loops + conditions be used together? | Yes                                           |
| Are `facts` used in conditions?          | Yes (e.g., OS type, CPU, memory)              |
