| Feature               | Use Case                      |
| --------------------- | ----------------------------- |
| `ignore_errors`       | Skip failure, continue        |
| `failed_when`         | Custom failure rules          |
| `rescue` block        | Try/Catch style handling      |
| `always`              | Always run cleanup/notify     |
| `any_errors_fatal`    | Fail fast for all hosts       |
| `max_fail_percentage` | Stop when too many hosts fail |
| `until`               | Retry until success           |
